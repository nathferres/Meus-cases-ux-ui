UX Design (ou Design de Experiência do Usuário) se concentra em criar experiências de usuário satisfatórias e significativas em produtos digitais, como sites, aplicativos, softwares e outros sistemas interativos. O objetivo do UX Design é proporcionar uma interação positiva, intuitiva e prazerosa para os usuários ao interagirem com um produto ou serviço.
Essa área é fundamental para garantir que cada pessoa tenha uma experiência eficiente e intuitiva ao usar um site, aplicativo ou qualquer outro sistema digital.
Em resumo, o UX Design é essencial para criar produtos e serviços que atendam às necessidades dos usuários, proporcionando uma experiência positiva e significativa durante a interação com o sistema.

Desenvolvimento web refere-se ao processo de criação e manutenção de aplicações web. Envolve a utilização de diversas tecnologias, linguagens de programação, frameworks e ferramentas para desenvolver e hospedar websites que possam ser acessados por meio de navegadores web.
Front-End: Lida com a interface do usuário e a interação direta com os usuários. Envolve HTML, CSS e JavaScript.
